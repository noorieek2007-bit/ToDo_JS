# JavaScript Todo List

A simple command-line Todo List application built with vanilla JavaScript.

## Features
- ➕ Add new tasks
- 📋 View all tasks
- ❌ Delete tasks by index
- 🚪 Quit the application
- ⚠️ Handles invalid commands and empty lists

## Concepts Practiced
- Variables
- Arrays
- Loops (`while`, `for`)
- Conditional Statements (`if`, `else if`, `else`)
- User Input (`prompt()`)
- Console Output (`console.log()`)
- Array Methods (`push()`, `splice()`)

## Technologies Used
- HTML5
- JavaScript (ES6)

## How to Run
1. Clone this repository.
2. Open `index.html` in your browser or use VS Code Live Server.
3. Open the browser console (`F12` → Console).
4. Follow the prompts to manage your todo list.
